# Staging Snapshots

This directory holds raw source snapshots captured before normalization.

Contents:

- `*.schema.json` — tracked field/provenance contract for each source (see
  `docs/dataset-spec.md`'s source-specific extraction contracts)
- `*.csv` — gitignored, regenerated by running an extractor:
  `python -m pipelines.cli extract <source>` (`pokeapi` | `opgg` |
  `munchstats`). These aren't committed: they grow/change on every refresh
  per `dataset-spec.md`'s refresh policy, and `munchstats.csv` alone is
  already ~37MB.

Release outputs should not be published from this directory directly —
normalization (`dbt/`) and validation happen first; see
`docs/dataset-spec.md`'s "Release package" section.

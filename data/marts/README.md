# Marts

Flat analytical exports and dashboard-ready summary tables (Phase 3, see
`docs/todo.md`), derived from `data/normalized/`'s core entities. Unlike
`data/normalized/`, these aren't locked entities from
`docs/dataset-spec.md`'s entity dictionary — they're derived/aggregated
views built for dashboard consumption.

Gitignored, regenerated by `make dbt-build` (`dbt/models/marts/`). See
`dbt/models/marts/schema.yml` for what each table contains and which
filter dimensions (regulation, date, Pokémon) are actually supportable
with current data versus `docs/prd.md`'s fuller dashboard ambition
(tournament tier, win-rate, move/item drill-down — not yet extracted).
